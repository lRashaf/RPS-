# 🎮 ElimiChaos

**ElimiChaos** is a stylish and interactive Rock Paper Scissors game built with **React**, **TypeScript**, and **Tailwind CSS**.

Challenge the computer in a fast-paced match where visuals meet simplicity. The game is sleek, responsive, and fun to play — perfect for showcasing your frontend skills!
